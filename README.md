# Introduction

Customer support ticket REST API microservices.

    - elasticsearch database storage for high speed full text queries
    - postgresql database
    - redis for instant token revocation
    - rabbitmq for asynchronous communication between services
    - spring cloud config for centralized configuration
    - spring cloud netflix eureka for service discovery
    - elk stack for centralized logging and visualization
    - docker for development infrastructure

This is my practice project for learning microservices. Work in progress.
